Number Guessing Game



Instructions:

  1. Guess the number by moving the slider.
  2. The color changes to yellow if the guessed number is smaller.
  3. The color changes to Red if the guessed number is greater.
  4. The color changes to Green if the guessed number is correct.
  5. You have a maximum of 5 attempts.
